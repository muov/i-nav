# ⚙️ DevOps 运维

## 服务面板

- [小皮面板](https://www.xp.cn/) - 好用安全稳定的服务器面板
- [1Panel](https://1panel.cn/) - 现代化、开源的Linux服务器运维管理面板
- [宝塔面板](https://www.bt.cn) - 安全高效的服务器运维面板

## SSH

- [WindTerm](https://github.com/kingToolbox/WindTerm) - 专业的跨平台 SSH/Sftp/Shell/Telnet/Tmux/Serial终端
- [Xterminal](https://www.terminal.icu/) - 更好用的开发工具，但不止于(SSH/控制台/More)
- [FinalShell](https://www.hostbuf.com/) - 一体化的的服务器，网络管理软件

## API 工具

- [Apifox](https://apifox.com/) - API 设计、开发、测试一体化协作平台
- [Apipost](https://www.apipost.cn/) - AI 驱动的下一代API开发管理工具
- [Reqable](https://reqable.com/zh-CN/) - 新一代API开发工具（抓包）

## 工具箱

- [UGit](https://ugit.qq.com/zh/) - 让每个人都可以轻松使用Git
- [ITDOG](https://www.itdog.cn/) - 在线ping、网站测速、路由追踪、DNS查询
- [爬虫工具库](https://spidertools.cn/#/) - 专为爬虫工程师打造的在线工具库

## 云平台

- [Netlify](https://www.netlify.com/) - 把你的想法推到网上
- [Vercel:](https://vercel.com/) - 使用 AI 云构建和部署最佳网络体验
- [Cloudflare](https://cloudflare.com/) - 全球分布式网络、安全防护及开发工具服务
- [CNB](https://cnb.cool/) - 基于 Docker 生态，云原生构建
